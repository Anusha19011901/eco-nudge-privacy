Privacy Policy – EcoNudge

Last updated: July 25, 2025

EcoNudge is a Chrome Extension designed to help users adopt more sustainable digital habits. Your privacy is important to us. This policy outlines how we handle data.

1. Information Collection

EcoNudge does not collect, store, or transmit any personally identifiable information (PII). All activity remains local to your browser.

We use Chrome’s storage API to save user preferences (e.g., selected categories for nudging) and interaction data (e.g., how many nudges you accepted). This data is stored locally and never sent to any server.

2. Permissions

EcoNudge uses the following permissions:

  • tabs and host_permissions to detect when you visit a page that may trigger an eco-nudge.
  • scripting and content_scripts to inject reminder banners.
  • storage to keep track of preferences and usage statistics.

3. Data Sharing

We do not share or sell any user data to third parties.

4. Remote Code / APIs

EcoNudge does not use any remote code or third-party scripts that execute in the user’s browser.

5. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected here.
